# B2B Business Marketing
### **Getting started**
### Technologies Used
#### HTML5, CSS3, Bootstrap 3.3, WOW.js, Animate.css, Font-awesome, Google Maps, Google Fonts
 - Run :  `git clone https://github.com/getwebem/b2bMarketing.git`
 - Run :  `cd b2bMarketing`
 - Run :  `Brackets .`
 - Open:  `Index.html`
 - Browser:  `Live Preview`
